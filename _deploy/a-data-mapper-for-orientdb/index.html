A Data Mapper for OrientDB

OrientDB

In DNSEE we’re getting serious about the CMF stuff.

So, next step, is to build a basic mapper for OrientDB.

If you want to partecipate to its development let me know it; the aim is to build a library doing the same things of Doctrine2, with annotations and so on.

Re-using LoCs of Doctrine2, of course.

The repo is on GitHub, and you’ll see a Mapper directory appear in the next days… ;–)
